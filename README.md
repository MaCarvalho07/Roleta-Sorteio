# 🎡 Roleta Sorteio

Projeto simples de roleta para sorteios aleatórios, ideal para escolher nomes, tarefas, prêmios ou qualquer tipo de decisão de forma rápida e visual.

---

## 📌 Sobre o projeto

A ideia é basicamente recriar aquele clássico de "girar a roleta" pra decidir algo — só que automatizado.

Esse tipo de ferramenta é comum pra:

* sorteio de nomes
* decisões em grupo (quem paga a pizza, quem faz tal coisa, etc.)
* brindes ou eventos
* uso em lives / streams

No geral, funciona com base em aleatoriedade, garantindo que todo mundo tenha a mesma chance. ([Central de Calculadoras][1])

---

## ⚙️ Funcionalidades

* 🎯 Adição de itens (nomes, opções, etc.)
* 🔄 Sorteio aleatório via roleta
* 🎡 Animação de giro
* 🧹 Limpeza/reset da lista
* 📊 Resultado exibido em tempo real

## 🚀 Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/MaCarvalho07/Roleta-Sorteio.git
```

2. Acesse a pasta:

```bash
cd Roleta-Sorteio
```

3. Abra o arquivo `index.html` no navegador

---

## 🧠 Como funciona

Você adiciona os itens na lista, clica pra girar e o sistema escolhe um resultado aleatório.
